# OpSem

> Experimental DSL for Operational Semantics, renders to LaTeX

For now, see the [tests](./src/test/java/OpSemLoaderTest.java) for usage. We can make a CLI if there's interest.

## Requirements
* Java 21

## Run

`./gradlew test`
